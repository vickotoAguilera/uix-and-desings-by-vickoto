# 01 - Visión del Proyecto

## Objetivo General
Crear una demostración interactiva de alto impacto visual ("UIX SHOWCASE PRO") que exhiba componentes de UI modernos con un enfoque especial en efectos de neón, glassmorphism, cyberpunk y estilos vanguardistas.

## Alcance Expandido
- **Estilos Sobrios y Corporativos**: Secciones limpias, minimalistas y profesionales para entornos de oficina/SaaS.
- **Especialización por Nicho**: Componentes dedicados a Cocina (Foodie UI), E-commerce (Shop UI) y Real Estate.
- Catálogo de +100 Hero Sections (expandiendo de 60).
- Catálogo de +100 Botones (Neon + Minimalistas).
- Catálogo de +100 Inputs y Formularios.
- Galería de fotos y efectos de imagen (Gradients, Glass, Parallax).

## Estéticas Objetivo
1.  **Cyberpunk/Neon**: Alto contraste, parpadeos, colores vibrantes.
2.  **Corporate Minimal**: Tipografía sans-serif limpia, mucho espacio en blanco, bordes redondeados suaves.
3.  **Modern SaaS**: Estilo "shadcn/ui", profesional, accesible.
4.  **E-commerce Vivid**: Colores cálidos, enfoque en producto, tarjetas de alta fidelidad.
5.  **Brutalist/Retro**: Diseños crudos, tipografía gigante, referencias a 8-bit/90s.
